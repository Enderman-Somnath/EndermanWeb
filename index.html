Security

The Database this website runs is Firebase.
The Database is password protected and only works with this website.
No other website can read or write to this website.
The Javascript is protected.
The Account is Protected by Firebase AUTH (means that password is not visible to anyone or the owner)
Different types of data may get collected as this website has Google ads.
Each message is encrypted with a unique key in ClassicChat room.


Safety
